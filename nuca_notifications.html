NUCA Notifications

Sistema de notificações estilizado para o site NUCA. Clique nos botões para testar:

Como usar no teu código:

// Substituir os alert() por:
showNotification('warning', 'Título', 'Mensagem');

// Tipos disponíveis:
// 'warning' - para campos obrigatórios (cor, tamanho, estação)
// 'error' - para erros
// 'success' - para confirmações
// 'info' - para informações gerais
