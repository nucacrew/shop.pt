Política de Privacidade

Última atualização: Março 2025

1. Quem Somos

A NUCA é uma marca de moda minimalista portuguesa, sediada em Lisboa. Somos uma loja 100% online e estamos empenhados em proteger a tua privacidade.

2. Dados que Recolhemos

Quando fazes uma compra ou nos contactas, recolhemos:

  • Nome e morada de envio
  • Email e telefone
  • Detalhes da encomenda

3. Como Usamos os Teus Dados

Utilizamos os teus dados apenas para:

  • Processar e enviar as tuas encomendas
  • Comunicar sobre o estado da entrega
  • Responder às tuas questões

Não vendemos nem partilhamos os teus dados com terceiros para marketing.

4. Proteção dos Dados

Os teus dados são guardados de forma segura e apenas acessíveis por nós. Utilizamos ligações encriptadas (SSL) para proteger as tuas informações durante as transmissões.

5. Os Teus Direitos

Tens o direito de:

  • Ver que dados temos sobre ti
  • Corrigir dados incorretos
  • Pedir a eliminação dos teus dados

Para exerceres estes direitos, contacta-nos: nuca.crew@gmail.com

6. Contacto

Para qualquer questão sobre privacidade:
